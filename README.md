# 🎓 Graphic Arts Institute Seat Planner (GAISP)

## 📌 Project Overview

Graphic Arts Institute Seat Planner (GAISP) is a web application that automatically generates **polytechnic-style seat plans** based on Department, Semester, Shift, Roll Range, Student List, Room Layout, and Seating Patterns.

It eliminates manual seat planning and generates a **ready-to-print PDF seat chart** following official institute formatting standards.

---

## 🚨 Problem Statement

Currently, teachers and office staff manually handle:

- Student roll arrangement
- Department mixing
- Room capacity calculation
- Seat chart creation
- PDF formatting and printing

This process is:
- Time-consuming ⏳  
- Error-prone ❌  
- Hard to manage at scale 📉  

GAISP automates everything.
